# Extimer - Exercise & Walking Tracker

A modern, clean web application for tracking daily exercise and walking activities with habit tracking capabilities.

## Features

- **Exercise Timer**: Track your workout sessions with customizable intervals
- **Walking Tracker**: Monitor your daily walks with duration and distance
- **Habits Tracker**: Build and maintain healthy habits
- **History & Analysis**: View your progress over time with date-wise tracking
- **Dark Mode**: Toggle between light and dark themes
- **Mobile Responsive**: Works seamlessly on all devices
- **Local Storage**: All data stored locally on your device

## Tech Stack

- **Vite** - Fast build tool and dev server
- **Vanilla JavaScript** - No framework dependencies
- **CSS3** - Modern styling with custom properties
- **Lucide Icons** - Beautiful icon library

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Clone the repository
```bash
git clone https://github.com/YOUR_USERNAME/extimer.git
cd extimer
```

2. Install dependencies
```bash
npm install
```

3. Run the development server
```bash
npm run dev
```

4. Open your browser and navigate to `http://localhost:5173`

## Build for Production

```bash
npm run build
```

The built files will be in the `dist` directory.

## Preview Production Build

```bash
npm run preview
```
